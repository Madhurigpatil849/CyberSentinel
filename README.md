🛡️ CyberSentinel: AI-Powered Threat Detection & Explainable Security

CyberSentinel is an advanced Android cybersecurity application that combines Artificial Intelligence, Real-Time Network Monitoring, Wi-Fi Security Analysis, Threat Remediation, and an LLM-Powered HelpBot to protect users from malicious applications, unsafe Wi-Fi networks, suspicious IP connections, and emerging cyber threats.

📊 High-Accuracy Neural Network Model trained on DREBIN, TUANDROMD, and COLCOM datasets

🎯 Problem Statement
Modern smartphones store sensitive information such as banking details, personal messages, and confidential data. Traditional antivirus applications rely primarily on signature-based detection and often fail to identify zero-day malware and network-based threats.
CyberSentinel addresses this challenge by combining AI-driven malware detection, real-time network analysis, explainable alerts, and intelligent user guidance into a single Android security platform.

🧠 Malware Detection Formula

0.5×Model Confidence+0.3×Category Risk+0.2×Sideloading Score

This contextual scoring mechanism improves accuracy and reduces false positives.

📈 Model Performance
🎯 Overall Accuracy: ~90%
📊 Macro F1-Score: 0.90
🟢 Goodware Recall: 0.95
🔴 Malware Precision: 0.93
🧪 Training Epochs: 27

The model demonstrated strong classification performance with low false positives and false negatives in experimental evaluation.
